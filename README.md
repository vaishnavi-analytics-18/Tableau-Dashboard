This dashboard provides an in-depth analysis of sales performance using multiple KPI and visualization sheets. It highlights key business insights through:

Key Features

KPIs Overview: Displays total Sales, Profit, Quantity, and Discount performance.

Sales Analysis: Visualizes sales trends across months to identify seasonality and peak performance periods.

Category & Sub-Category Insights: Shows which product categories and sub-categories contribute the most to revenue.

Top 5 Products: Identifies best-performing products based on sales.

Profit Margin Analysis: Highlights most and least profitable product segments.

State-Level Performance: Provides geographical insights to understand regional contribution.

Return Rate Analysis: Monitors return percentage to evaluate product performance quality.
